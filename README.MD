## 安裝流程
請依照敘述於終端機輸入指令

1.開啟終端機將專案存至本機:
```
git clone https://github.com/alan78814/AC_S2-3_Restaurants-List.git
```
2.進入專案目錄
```
cd AC_S2-3_Restaurants-List
```
3.安裝套件
```
npm install
```
4.加入種子資料
```
npm run seed
```
5.啟動專案
```
npm run dev
```
6.可開啟瀏覽器輸入http://localhost:3000 開始使用
